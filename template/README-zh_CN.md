## 介绍

该模板工程为 React Native 0.59 版本的 TS 基础模板，可启动所有涂鸦产品，用于调试当前产品的功能点，此外支持以下功能：

1. Path Alias;
2. React、Redux Hooks;


## 快速运行

```bash
$ npm install && npm run start
# 或者
$ yarn && yarn start
```
